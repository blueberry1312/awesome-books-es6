# 📖 [Awesome-books-es6] <a name="about-project"></a>


**[Awesome-books-es6]]** Awesome books is a basic website that allows users to add/remove books from a list. In this project, we will restructure the Awesome books app code. The goal is to make it more organized by using modules. We will also practice the ES6 syntax learned.
 

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

HTML, CSS, JavaScript

### Key Features <a name="key-features"></a>

- **[add_book]**  Button to add book.
- **[remove_book]** Button to remove book.
